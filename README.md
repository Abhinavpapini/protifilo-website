## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm or bun package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Abhinavpapini/protifilo-website.git
cd protifilo-website
```

2. Install dependencies:

3. Start the development server:
